# 💸 Splitter - Modern Tip Calculator

A sleek, responsive tip calculator built with a modern tech stack. This project was refactored from a vanilla JavaScript challenge to a robust, type-safe, and highly maintainable React application.

![Design Preview](./design/desktop-preview.jpg)

## 🚀 Features

- **Real-time Calculations**: Instant updates for tip amount and total per person.
- **Dynamic Tip Selection**: Choose from predefined percentages or enter a custom amount.
- **Responsive Design**: optimized for both mobile and desktop viewing.
- **Input Validation**: Ensures calculations only run when all necessary fields (Bill, People, and Tip) are provided.
- **Clean UI**: Built with Tailwind CSS for a premium, modern feel.

## 🛠️ Tech Stack

- **React 19**: For building the interactive user interface.
- **TypeScript**: For robust type safety and better developer experience.
- **Tailwind CSS**: For utility-first, responsive styling.
- **Vite**: For a lightning-fast development environment and optimized builds.

## 📥 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/habibatsulayman/Tip-Calculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Tip-Calculator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## 💻 Usage

To start the development server:
```bash
npm run dev
```

To build the project for production:
```bash
npm run build
```

## 🌐 Deployment

This project is ready to be deployed on **Netlify** or **Vercel**. 
- **Build Command**: `npm run build`
- **Publish Directory**: `dist`
